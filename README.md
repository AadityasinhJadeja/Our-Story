# About Us

Voice-first romantic storytelling prototype.

## Run

Open `/Users/aadityasinhjadeja/Documents/New project/index.html` directly in a modern browser, or run a local static server:

```bash
cd "/Users/aadityasinhjadeja/Documents/New project"
python3 -m http.server 4173
```

Then visit `http://localhost:4173`.

## What is implemented

- Premium editorial landing page with subtle petal motion and chapter previews.
- Guided chapter-by-chapter storytelling flow (one prompt at a time).
- Voice capture using Web Speech API (with demo fallback).
- Wispr-style transcript rendering and completion confirmations.
- Visual personalization engine that maps story nouns to motifs (cake, beach, rain, coffee, stars, college/books).
- Artifact renderer that generates a magazine-style digital love story:
  - Cover page
  - Narrative spread
  - Timeline
  - Love DNA
  - Little Things cards
  - Inside Dictionary
  - Promise climax
  - Closing page
- Export and sharing controls:
  - Shareable URL with encoded responses in hash
  - Print-to-PDF mode
  - Screenshot mode
